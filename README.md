# Hostel Management Server

This is the backend server for the AI-Powered Hostel Management System.

## Project Overview
The server application is built with Node.js and Express, providing APIs for the hostel management system.

## Technologies Used
- **Node.js & Express**: Backend framework
- **Mongoose**: MongoDB object modeling
- **JWT**: Authentication
- **BcryptJS**: Password hashing
- **Nodemailer & SendGrid**: Email services
- **Multer & Cloudinary**: File uploads
- **Razorpay**: Payment processing
- **PDFKit**: PDF generation
- **Google GenAI**: AI integration
